# yixiangshetuan #
# 意想社团 #


项目名：意想社团

来源：大三下学期node课程设计

方式：组队

框架：angular2 + express

后台：nodejs

数据库：mysql


由于时间匆忙，本项目并未使用webpack或其他前端打包框架，通过.ejs模板框架渲染底层页面，angular访问接口渲染数据。

纯接口开发模式，数据的往来都通过接口JSON结构数据。
